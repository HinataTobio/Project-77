Project on console

Visual Studio Code is a code editor made by Microsoft for Windows, Linux and macOS. Features include support fordebugging, syntax highlighting, intelligent code completion, snippets, code refactoring, and embedded Git.

The console. log() method outputs a message to the web console. The message may be a single string (with optionalsubstitution values), or it may be any one or more JavaScript objects. Note: This feature is available in Web Workers.
